🗑️ VK Messenger Chat Cleaner

A powerful DevTools script to permanently delete all chat histories from your VK profile.

✅ Setup Complete! Your cleaner is ready to use.

Quick Start (3 Steps)

Step 1: Copy the Script

Open the file: dist/devtools-script.js

Select all code (Ctrl+A) and copy it.

Step 2: Open DevTools

Go to VK.com → Press F12 → Click Console tab

Step 3: Paste & Run

Paste the code into console and press Enter, then type:

startVKCleaner()

How It Works

Step What Happens
1. Validate Checks if you're logged into VK and API is accessible
2. Fetch Retrieves all your conversation IDs with pagination
3. Confirm Shows dialog asking for final confirmation
4. Delete Permanently deletes each conversation (350ms apart)
5. Report Shows final statistics and execution time

Important Warnings

⚠️ IRREVERSIBLE! Deleted conversations cannot be recovered. Think twice before confirming deletion.
⚠️ NO BACKUP! This permanently removes all chat histories. There's no undo button.

Troubleshooting

Problem Solution
"Cannot access VK API" Log in to VK.com first, then refresh the page
Script doesn't appear Make sure you pasted in the Console tab, not elsewhere
Deletion is very slow This is normal - rate limiting protects your account
"Too many requests" error VK rate limiting - script will retry automatically

FAQ

Q: Will this affect my VK profile?

No, only chat messages are deleted. Your profile, friends, posts stay intact.

Q: Can deleted messages be recovered?

No, deletion is permanent and irreversible.

Q: Will VK ban my account?

No, this uses the official public API with normal rate limits.

Q: How long does it take?

Approximately 350ms per conversation (e.g., 100 chats = ~35 seconds).

Q: Do I need an API token?

No, it uses your existing browser session authentication.
